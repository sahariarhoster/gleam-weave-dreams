Send WhatsApp notifications to customers and admins on WooCommerce order events via the WA Notifier panel.

== Description ==
Activate with a license generated in your WA Notifier brand owner portal. Choose a device, configure per-status customer and admin templates with shortcodes ({first_name}, {last_name}, {order_id}, {total}, ...) and send test messages — all from a guided 5-step wizard.

== Installation ==
1. Upload the plugin folder to /wp-content/plugins/.
2. Activate the plugin.
3. You will be redirected to the setup wizard. Paste your license key and follow the steps.
